# Hotel Management System

A simple **Hotel Management System** implemented in **C++** as a practical project for the **Data Structures** course.

## Description
The system manages hotel room bookings using a **Doubly Linked List**.  
Each room record contains:
- Room number
- Guest name
- Room type (Single / Double / Twin)
- Check-in date
- Stay duration

## Features
- Register a new room
- Search for a room by room number
- Update room information
- Delete a room record
- Display all registered rooms

## Data Structure Used
- **Doubly Linked List**
  - Allows easy insertion, deletion, and traversal of room records.

## Technologies
- Language: **C++**
- Concepts: Linked Lists, Classes, Pointers, Input Validation
